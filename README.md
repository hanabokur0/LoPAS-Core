# LoPAS-Core v1.3

The core reference implementation of the **LoPAS (Local-Personal AI System)** indicators,
evaluation pipeline, and API specification.

This repository will host:
- Cloudflare Worker source code for the LoPAS Core API
- `/v1/eval` text analysis endpoint
- `/v1/score/*` indicator calculators
- Full JSON Schema for all primary and cognitive indicators
- Examples, demo notebooks, and PoC templates
